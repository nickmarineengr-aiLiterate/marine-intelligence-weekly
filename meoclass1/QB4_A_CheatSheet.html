📋 Document Issuance & Inspectability Map

ISM — DOC / SMC

  • DOC → Company (certified copy onboard, original ashore)
  • SMC → Ship directly
  • Loss of DOC → SMCs invalid → ship cannot trade

ISPS — ISSC / SSP

  • ISSC 5yr, intermediate 2nd–3rd yr
  • Interim ISSC max 6mo, non-renewable, no 2 consecutive for same reason
  • SSP = CONFIDENTIAL — PSCO sees ISSC + DoS, not SSP text
  • Security level set ONLY by Contracting Government

MLC — DMLC I & II

  • Part I → Flag State (the laws)
  • Part II → Company, Flag-approved (the measures)
  • Both PSC-inspectable (unlike SSP)
  • 16 items now: 14 core + 2 financial security (2014 amdt)

PSC vs FSI vs Internal Audit

  • FSI issues/suspends certs (UNCLOS Art 94)
  • PSC detains, cannot cancel certs (UNCLOS Art 218/219)
  • ISM Internal Audit raises "non-conformities" — separate from PSC "deficiencies"

Flag State vs RO

  • Flag State delegates functions, never responsibility (UNCLOS Art 94)
  • RO = e.g. IRS for Indian-flagged vessels, authorised under SOLAS Ch I/Reg 6

Class Notations

  • Recommendation = non-binding
  • CoC = binding, due date, suspension if missed
  • Immediate CoC = before sailing
  • Suspension → ALL RO-issued statutory certs invalid simultaneously + P&I cover typically void
