3Bulk Cargo Risk Matrix

The single most-confused table in oral exams. Memorise cargo → main risk → governing code.

CargoMain RiskGoverning Code
Nickel OreLiquefactionIMSBC (Group A)
BauxiteDynamic separationIMSBC (Group A — special schedule)
CoalSelf-heating + liquefaction (dual: Group A & B)IMSBC
GrainCargo shift / heeling momentInternational Grain Code
Iron OreHigh density — structural (SF/BM)IMSBC (heavy cargo provisions)
Timber / LogsDeck cargo shift + weight gainTDC Code (CSS Code framework)
Examiner trap: Bauxite and Nickel Ore are both Group A liquefaction risks but bauxite has its own special IMSBC schedule (dynamic separation — wider particle size distribution behaviour). Coal is the only cargo that can be BOTH Group A and Group B simultaneously. See page 13 for full Coal/Ore/Grain comparison.

4IMSBC Group A / B / C Flowchart

A must-have. Every bulk cargo question starts by placing the cargo in one of these three groups.

IMSBC CODE
Group A
→ Liquefaction risk
  • Nickel ore
  • Bauxite (fines)
  • Iron ore fines
  • Coal (some grades)
Key check: MC < TML before loading
Group B
→ Chemical hazard
  • Coal (methane/self-heating)
  • DRI (direct reduced iron)
  • Sulphur
  • Ammonium nitrate-based fertilizers
Key check: gas monitoring / ventilation regime
Group C
→ Neither A nor B
  • Grain
  • Steel coils/plates
  • Cement (bulk)
  • Most general bulk solids
Key check: still apply general IMSBC stowage/segregation rules
Coal is the exam favourite: it can be listed under A AND B at the same time — liquefaction (A) and self-heating/methane (B) are independent hazards requiring independent checks. Never say a cargo is "only Group A" without checking its full IMSBC schedule entry.

5Liquefaction Decision Tree

Extremely important — this sequence is the practical heart of Group A cargo acceptance.

CARGO RECEIVED FOR LOADING
↓
Group A cargo?
↓ YES
TML / Moisture Certificate available?
Yes — MC < TML
Proceed with loading. Re-confirm MC if cargo condition changes (e.g. rain exposure) during loading.
No / MC ≥ TML
STOP. Do not load. CE/Master refuse cargo until a valid certificate confirms MC < TML. This overrides commercial/schedule pressure.
Rain-during-loading sub-case:
Rain falls on stockpile / open holds during loading
↓
Re-test moisture content before resuming/completing loading
TML = 0.90 × Flow Moisture Point. The CE's role is verification, not testing — but refusal authority is real and exam-tested (see page 15, Red Lines).

8Container Ship Safety Timeline

Shows the evolution of container safety regulation — a strong "modern topic" narrative for Nair/Simon.

Problem: Lost Containers at Sea
↓ led to

↓ enabled by
Container Tracking technology (RFID/GPS/IoT)
↓ alongside
Fire Detection improvements (Li-ion driven)
↓ culminating in

Narrative for oral exam: "Each safety gap exposed by an incident has driven the next regulatory step — VGM responded to weight-related loss/stability incidents, tracking responded to the difficulty of locating lost units, and current SOLAS work responds to the fire risk profile of modern cargo (especially Li-ion)." This kind of cause→regulation framing impresses examiners.

10IBC Ship Type Diagram

Candidates often remember the numbers (1/2/3) but not the underlying reason — survivability after damage.

TypeHazard LevelSegregation / SurvivabilityWhy
Type 1HighestMaximum — cargo tank survives most severe damage (collision/grounding)Cargo poses severe hazard to life/environment if released
Type 2MediumModerate — survives standard damage extentSignificant but lesser hazard than Type 1
Type 3LowerLeast — minimum location requirementsCargo presents sufficiently minor hazard that minimal survivability suffices
The "why" examiners actually want: the ship type number determines how far inboard / how well-protected the cargo tank must be — i.e. the ship's damage survivability requirement scales with the cargo's hazard to people and the environment if the tank is breached. Type 1 = tank positioned for maximum protection; Type 3 = least restrictive.

11Stability Danger Tree

A great oral revision sheet — ties together every stability-loss mechanism in QB2 under one root.

LOSS OF STABILITY
Liquefaction
Free surface effect → sudden large-angle list → possible capsize. (Group A cargo, MC ≥ TML)
Grain Shift
Assumed heeling moment from void migration → sustained heel, reduced residual stability/freeboard.
Parametric Rolling
Wave-period/roll-period resonance (container ships) → large roll angles → cargo/lashing failure, container loss.
Common Monitoring & Control Tools
Loadicator
Verifies GM/SF/BM at every loading stage (links to page 9 BLU+IMSBC synthesis, QB2 Q29).
GM Management
Liquefaction (FSE drop), Grain (corrected GM ≥ 0.30m), Timber (KG rises with weight gain) — GM is the common thread.
Cargo Securing
Lashing/shifting boards/bagging — physical prevention of shift across grain, timber and container cargoes.
If asked "what do these three have in common?" — answer: all three reduce effective GM or create an unplanned heeling/rolling moment, and all three are mitigated through loadicator verification + correct cargo securing.

12Parametric Rolling — Visual

Simple and memorable — a frequent container-ship cross-question (Simon/Nair).

Wave Encounter Period ≈ Natural Roll Period (ratio ~1:2 or 1:1)
↓
Resonance — GM varies cyclically as ship pitches through wave crests/troughs
↓
Large, rapidly-building Roll Angles (can exceed 30° within a few cycles)
↓
Container/Lashing Failure → Container Loss Overboard
Avoidance (master/CE action): change course and/or speed to break the resonance condition — alter wave encounter angle away from head/stern seas, typically toward beam-on headings is avoided too; the goal is to shift encounter period away from the roll-resonance band. Particularly relevant to large container ships in head/quartering seas at specific speeds.

15Chief Engineer's Red Lines — "Do Not Load If..."

A famous MIW page in the making. Examiners love practical judgment over textbook recitation. Maps directly to QB2 Q30.

CargoRed Line — Will NOT Load If
Nickel Ore / Group ANo TML on record / no Moisture Certificate
Water observed on stockpile prior to loading
Rain during loading and moisture not re-tested before resuming
CoalHigh methane reading or elevated temperature trend without ventilation/monitoring plan
GrainNo approved stability calculation / no valid Document of Authorisation (all 3 Grain Code criteria not met)
ChemicalCargo not listed in IBC Code Chapter 17 / not on ship's ICOF / MSDS shows tank incompatibility
ContainerUnknown or undeclared DG content (mis-declaration suspected), or no VGM
Timber20–30% weight-gain effect not accounted for in voyage stability plan
Delivery tip: present 3–4 of these as confident "I will not proceed if…" statements, each tied to a specific code/convention. If the examiner pushes with "but the charterer insists" — statutory/class requirements are not commercially negotiable. Full detail in QB2 Q30.
