Topic 30

Port State Control (PSC) — Compliance, Deficiency Categorization & Detention Appeal Protocols

PSC / Statutory Compliance Uday pp. 151–159 Examiner: Nair ⭐ High Frequency
🔎 Verification note: The source draft correctly identifies Resolution A.1185(33) (adopted 6 December 2023, effective 1 January 2024) as governing PSC procedures, and correctly states its Action Code matrix (Code 30 = detainable, Code 70 = RO informed, Code 17 = rectify before departure, Code 15 = rectify at next port) — these match the verified figures on file. However, this instrument has since been superseded. The IMO Assembly adopted Resolution A.1206(34) — Procedures for Port State Control, 2025 on 3 December 2025, which explicitly revokes A.1185(33) and took effect from 1 January 2026. As of this writing (July 2026), A.1206(34) is the current governing instrument. The core Action Code numbering and detainable-deficiency logic described below are carried forward unchanged into A.1206(34); the principal changes are a new Appendix 20 (dedicated PSCO guidelines on ISPS/security aspects, working with a Duly Authorized Officer/DAO), consolidation of detainable-deficiency criteria into Appendix 2, and clearer linkage of LRIT non-compliance to detention. All reg-box entries and CE Oral Tips below have been updated to cite A.1206(34) as current, with A.1185(33) retained parenthetically for historical/appeal-document continuity. Candidates should be prepared for either citation to be accepted orally at Kochi MMD during the 2026 transition period, but should lead with A.1206(34) as the current instrument.

📖 Definition

Port State Control (PSC) is the internationally mandated inspection regime under which a sovereign coastal/port State exercises jurisdiction over foreign-flagged vessels calling at its ports, to verify that the ship's condition, equipment, manning, and operational management comply with the international maritime conventions to which that State is a Party. PSC functions as the "second line of defence" behind Flag State control and Classification Society (RO) oversight — it exists precisely because Flag State self-regulation, especially under flag-of-convenience registries, has historically proven insufficient on its own.

⚓ Why It Matters (CE Perspective)

  • A single PSC detention converts directly into off-hire losses, charter-party penalties, and adverse targeting factor scoring under the relevant regional MoU — the CE's watchkeeping and maintenance standard is under continuous, unannounced audit every time the vessel enters a foreign port.
  • Sub-standard vessels distort freight markets by undercutting compliant operators; PSC is the enforcement mechanism that keeps that commercial playing field level, which is why underwriters, P&I clubs and charterers all track detention history.
  • Since A.1185(33) (carried into A.1206(34)), missing EEXI documentation, an invalid SEEMP, or absent CII compliance confirmation are explicit detainable deficiencies — GHG/decarbonisation compliance is no longer a "soft" administrative matter for the CE, it is a detention trigger.
  • A Code 30 (detainable) deficiency legally compels the PSCO to issue a Notice of Detention; the CE's technical judgement in the hours before a PSC inspection materially affects whether a defect is logged as Code 16/17 (rectifiable) versus Code 30 (detainable).

🔧 Deficiency Classification & the Mathematics of Targeting

ClassDefinitionOperational Consequence
MinorIsolated issue, no immediate systemic threatVessel sails; rectify within stated timeline (Code 15/16)
MajorSignificant non-conformity compromising safety systems or environmental barriersImmediate corrective action required before proceeding (Code 17)
DetainableSevere, systemic failure — objectively unseaworthy or unacceptable risk to life/safety/environment; explicitly includes missing EEXI, invalid SEEMP, or absent CII compliance confirmationMandatory Notice of Detention (Code 30) — vessel restrained until rectified or conditional voyage approved

Regional MoUs calculate a vessel's Ship Risk Profile (SRP) / Targeting Factor using a weighted risk model aggregated over a rolling window (Paris MoU uses inspection windows of roughly 5–6 months for High-Risk Ships, 10–12 months for Standard-Risk, and 24–36 months for Low-Risk Ships; Tokyo MoU aligns closely but is not identical, and the USCG operates an entirely independent matrix boarding priority system). The generic weighted-factor structure is illustrative of how age, flag performance, RO performance, company performance and historical deficiency record combine — exact weighting coefficients are internal to each MoU's targeting algorithm and are not published as a fixed public formula, so treat any specific numeric weighting as conceptual rather than an examinable exact figure.

🎯 Oral Q&A

Q1: What is Port State Control and why does it exist alongside Flag State control?
PSC is the inspection authority a coastal/port State exercises over foreign-flagged vessels calling at its ports, verifying compliance with international conventions. It exists as a "second line of defence" because Flag State self-regulation — particularly under flags of convenience — has historically been inconsistent; PSC prevents sub-standard vessels from simply routing to ports with lax enforcement ("port shopping").
Q2: Which resolution currently governs PSC procedures, and what did it replace?
Resolution A.1206(34), Procedures for Port State Control 2025, adopted 3 December 2025 and effective from 1 January 2026. It explicitly revokes the previous instrument, A.1185(33) (2023). The underlying Action Code matrix and detainable-deficiency logic carried forward unchanged; the 2025 revision's main addition is a dedicated Appendix 20 on ISPS/security inspection guidance.
Q3: What is the difference between Code 17 and Code 30?
Code 17 (rectify deficiency before departure) applies to a major deficiency that must be resolved and PSCO-verified before the vessel can unmoor, but does not by itself arrest the vessel's legal status. Code 30 (detainable deficiency) is a formal Notice of Detention — a severe, systemic failure rendering the vessel objectively unseaworthy, legally restraining it from departing at all until the defect is rectified or a conditional voyage is approved.
Q4: Give three examples of detainable deficiencies under A.1206(34)/A.1185(33) relevant to a Chief Engineer.
Absence of a valid required EEXI technical file, absence of a valid SEEMP, and absence of the required Confirmation of Compliance for annual CII/fuel-oil-consumption reporting are all explicit detainable deficiencies. Traditional detainable items also include failure of main/auxiliary steering gear and serious safety-equipment deficiencies.
Q5: What triggers Code 70, and why does it matter to the CE?
Code 70 means "Recognized Organization informed" — the PSC authority formally notifies the issuing Classification Society of a structural or statutory system failure found during inspection. It matters because it typically follows an RO's own statutory certificate being called into question, which may trigger an emergency class survey and possible suspension of the certificate the RO issued.
Q6: What was the historical catalyst for regional PSC cooperation?
The Amoco Cadiz grounding off Brittany, France, in 1978, which caused catastrophic oil pollution and exposed the weakness of fragmented, individual national enforcement. This directly led to 14 European States signing the Paris MoU in 1982, the first and most influential regional PSC agreement, which prevented sub-standard vessels from "port shopping" to weaker-enforcement ports.
Q7: How many regional PSC regimes exist globally, and is the USCG one of them?
There are nine PSC regimes worldwide — eight regional MoUs plus one Agreement (Viña del Mar). The USCG (United States) operates its own independent PSC/boarding-priority matrix and is not a member of any of the nine regional MoUs, though it cooperates informationally with them.

Appeal Protocols

Under both A.1185(33) and A.1206(34), a master/owner has the right of appeal against a detention decision. The PSCO must inform the master in writing of this right at the time of detention. Appeal is first directed to the detaining Authority; if unresolved, it may be escalated per the relevant regional MoU's own appeal procedure. Undue detention or delay entitles the owner to compensation for resulting loss or damage — this is an explicit Procedures provision, not merely a commercial expectation.

What Changed in A.1206(34) Specifically

The 2025 revision consolidated detainable-deficiency criteria into a single Appendix 2 (previously scattered across several appendices covering ISM, LRIT, STCW and MARPOL Annex VI items), added a dedicated Appendix 20 on security/ISPS inspection guidance involving a Duly Authorized Officer (DAO) working alongside the PSCO, and more explicitly linked LRIT non-compliance to detention (previously often treated as an administrative-only matter).

On My Vessel (Maersk container, 2-stroke main engine)

For a Maersk-managed container vessel under Danish or similar flag administration, PSC exposure is highest in high-traffic Paris MoU and Tokyo MoU ports. The CE's practical defence is ensuring EEXI/SEEMP/CII documentation folders are current and immediately producible, main/auxiliary steering gear test logs are up to date, and any outstanding Condition of Class is disclosed proactively rather than discovered by the PSCO — concealment converts a Code 17 into a Code 30 and a Code 70 RO referral.

Classification Societies — Statutory Mandates, In-Service Surveys & Class Status Lifecycle

Class / Survey Regime Uday pp. 160–167 Examiner: Paul ⭐ High Frequency
🔎 Verification note: The source draft's survey-interval figures (Annual ±3 months, Special/ Renewal every 5 years, Bottom/Docking twice in 5 years with max 36-month out-of-water interval, Boiler twice in 5 years with max 36-month internal interval) are consistent with the IACS-harmonized survey framework applied across major societies and IMO's HSSC (Harmonized System of Survey and Certification) guidelines, so these are retained as stated. The 6-month automatic-suspension trigger for a lapsed CoC extension request is standard IACS practice (PR1) and is retained.

📖 Definition

A Classification Society is an independent, non-governmental technical organization that establishes and maintains standards for the structural design, construction, and operational-lifecycle fitness of ships. Class verifies the structural integrity of the hull, propulsion, steering, and electrical power-generation systems through continuous plan approval and periodic in-service surveys, giving owners, underwriters, charterers, and Flag States an objective, independently audited standard of seaworthiness.

⚓ Why It Matters (CE Perspective)

  • A valid Certificate of Class is a strict prerequisite for Hull & Machinery and P&I insurance — no underwriter will cover a vessel without active class, so a CE's survey preparation directly protects the ship's commercial insurability.
  • Class acts as a Recognized Organization (RO) delivering statutory certificates (SOLAS, MARPOL, Load Line) on behalf of the Flag State — the CE deals with the same surveyor for both class rules and statutory compliance in most inspections.
  • An overdue or mismanaged Condition of Class (CoC) is a direct PSC detention trigger (Code 30) — CoC tracking is a CE-level responsibility, not purely an office/DPA function.
  • Any unapproved structural or machinery modification made without class drawing approval and surveyor attendance can trigger automatic suspension of class — this includes emergency engineering fixes made under operational pressure at sea.

CE Oral Tip — Paul Sir: Paul examines the precise legal distinction between a Condition of Class (issued by the Society against its own Class Rules) and a Statutory Exemption (issued by/on behalf of the Flag State against convention requirements) — candidates who conflate the two are marked down immediately. 🎯 Oral Q&A

Q1: What is the difference between a Condition of Class and a Statutory Exemption?
A Condition of Class (CoC) is issued exclusively by the classification society when hull or machinery has a defect violating the society's own Class Rules but not posing immediate danger — it carries an operational limitation and a rectification deadline. A Statutory Exemption is issued by or on behalf of the Flag State Administration, providing a legal relaxation from mandatory convention requirements (SOLAS/MARPOL). Class cannot grant statutory exemptions unless specifically authorized as an RO for that vessel by the Flag State.
Q2: Under what conditions is a vessel's class automatically suspended?
Automatically, if the Class Renewal/Special Survey is not completed by its due date without an approved extension, or if Annual/Intermediate Surveys are not completed within their mandatory windows (±3 months of the anniversary date). It can also be compromised if the vessel sails with a known unrectified structural/mechanical defect without notifying class.
Q3: What is the role of Class during a major conversion (e.g., tanker to bulk carrier)?
Class acts as the structural control framework throughout: the owner submits engineering drawings, scantling calculations, and stability modifications for plan approval before alteration begins; surveyors maintain continuous shipyard attendance to verify materials, witness welding, monitor NDT; on completion, Class updates stability data, issues a modified Certificate of Class, and performs the corresponding statutory RO surveys.
Q4: What is the maximum interval between out-of-water Bottom/Docking Surveys?
36 months — the survey must occur twice within any 5-year class cycle, with no more than 36 months elapsing between two consecutive out-of-water inspections.
Q5: Trace the historical origin of classification societies.
The concept originated in the mid-18th century at Edward Lloyd's Coffee House in London, where marine underwriters assessed ship quality for insurance purposes. The first independent shipping register was published in 1764 by the Society for the Registry of Shipping, evolving into Lloyd's Register. Hulls were originally graded with vowels and equipment/manning with letters — the designation "A1" (first-class hull, excellent equipment) became the global benchmark phrase still used informally today.

Case Study — Localized Hull Wastage via Jetting Collision

A Capesize bulk carrier strikes a jetty fender during berthing, causing a 1-metre crack in side shell plating above the waterline, compromising Topside Ballast Tank No. 4 Port. Response sequence: (1) CE/Chief Officer isolate the tank and map crack extent; (2) Master notifies Class and Flag State, requesting emergency survey; (3) surveyor assesses — permanent hot-work repair often cannot be done at a cargo berth for port-safety reasons; (4) surveyor issues a CoC with an operational deadline (illustrative example in the source draft: 60 days) and interim restrictions (e.g. not fully loading the affected tank, daily visual boundary checks); (5) at the repair yard, an insert-plate repair is executed, NDT witnessed by the surveyor, and the CoC deleted from the register on satisfactory close-out.

On My Vessel (Maersk container, 2-stroke main engine)

If the main engine governor fails at sea with no approved spare available, an unapproved mechanical bypass must never be fabricated — this invalidates propulsion safety redundancy and risks automatic class suspension if discovered without surveyor authorization. Correct procedure is to notify the company and arrange an emergency class survey attendance at the next port, keeping the vessel's operational limitation documented in the deck/engine log in the interim.

The IACS Framework — Unified Requirements, Class Transfer & Multi-Class Operations

IACS / Class Transfer Uday pp. 168–175 Examiner: Paul Medium Frequency

📖 Definition

The International Association of Classification Societies (IACS) is a non-governmental organization uniting the world's leading classification societies to establish, review, and harmonize technical standards for ship construction and operational safety. Its member societies collectively class the substantial majority of world commercial cargo-carrying tonnage — commonly cited at over 90% — making IACS the effective global technical backbone beneath IMO's statutory framework.

⚓ Why It Matters (CE Perspective)

  • IACS Common Structural Rules (CSR) directly set the wave-load, fatigue-life, and net-scantling formulas that determine hull renewal thresholds a CE will see quoted in thickness-gauging reports.
  • During a Transfer of Class (TOC), the CE is the primary custodian of survey history, CoC records, and thickness-gauging logs — concealing an outstanding recommendation from the incoming society is a direct PR1A violation with immediate consequences for the interim certificate.
  • Under a Dual/Double Class arrangement, if one society suspends class the CE must understand that the remaining society is obliged to suspend without delay — there is no "continuing to trade under the other class" workaround.
  • IACS's Early Warning Scheme (PR8) is how systemic design/material failures (e.g., a specific crankshaft or turbocharger rotor casing defect) get flagged fleet-wide — a CE's incident report can trigger a global design-rule update.

🎯 Oral Q&A

Q1: What is IACS and when was it founded?
The International Association of Classification Societies (IACS) is a non-governmental body uniting the world's leading classification societies to harmonize technical shipbuilding and safety standards. It was formally founded in 1968 by seven leading societies, following preliminary conferences in 1939 and 1955, and was granted consultative status with IMO in 1969. It currently has 12 member societies.
Q2: Explain the timeline and data exchange required under PR1A when a class transfer request is made.
Within two working days of receiving the owner's request, the Gaining Society (GS) must notify the Losing Society (LS) and request the vessel's complete current class status. The LS must transmit the full survey history, all outstanding Conditions of Class, any operational restrictions, and complete UT records. The GS cannot issue an Interim Certificate of Class until this technical data package has been received, reviewed and integrated into its registry.
Q3: What additional requirement applies to a vessel 15 years or older undergoing a class transfer?
In addition to closing out all overdue surveys and outstanding Conditions of Class, the Gaining Society must complete a specialized class entry survey matching the scope of an Annual or Renewal Survey (depending on the vessel's specific status) before an Interim Certificate can be issued — this is materially more stringent than the under-15-year threshold.
Q4: What is the engineering purpose of the IACS Early Warning Scheme (PR8)?
PR8 requires any IACS member identifying a serious or repetitive technical failure — e.g., a specific main engine crankshaft cracking pattern, recurring propeller blade detachment, or localized bulkhead buckling — to file an incident report shared across all member societies. This lets societies instantly update design rules, increase inspection frequency for the affected component, and issue fleet-wide safety alerts.
Q5: Distinguish an IACS Unified Requirement (UR) from a Unified Interpretation (UI).
A UR is a standardized technical resolution setting a mandatory minimum design/construction/survey requirement — member societies may impose stricter rules but not weaker ones. A UI clarifies ambiguous language within an existing IMO convention so all member societies apply the same interpretation; it does not itself set a new technical minimum, it harmonizes reading of an existing one.

PR36 — Flag State Change Coordination

When a vessel changes its Flag Administration (as opposed to a straightforward class transfer), IACS PR36 requires the gaining and losing societies to coordinate transfer of all historical statutory documentation, so the new Flag State receives full visibility of the vessel's technical history — this runs in parallel with, and is distinct from, the PR1A class-transfer data package.

PSC Scrutiny of Recent Transfers

Port State Control Officers pay close attention to vessels that have recently undergone a Transfer of Class. If Form A/Form B review reveals the transfer occurred while detainable deficiencies were active, the vessel is flagged for expanded inspection — all historical close-out documentation from the losing society should be verifiably stamped, logged, and signed off by the incoming surveyor before sailing.

On My Vessel (Maersk container, 2-stroke main engine)

On a Maersk-operated container vessel classed by a major IACS member, any tailshaft or hull thickness gauging record discrepancy discovered during a class transfer review would fall directly on the CE's record-keeping. Proactive, complete documentation — rather than waiting for the incoming surveyor to find gaps — is the practical defence against transfer delays and PSC expanded-inspection flags.
