Examiner Profiles — Kochi MMD
Nair
Conventions & MARPOL
Demands exact regulation numbers and resolution dates. Cross-questions on PSC action codes, ISM audit trails, and flag state obligations. Frequently visits container vessel scenarios.
Rajappan
Stability & Cargo
GZ curve, IS Code criteria, dynamic vs static stability. On tankers: IBC Code, cargo residue categories, loading manual compliance. Tests casualty-linked regulatory changes.
Srivastava
Electrotechnology
Circuit-level questions, controller tuning, UMS alarm logic, electrical protection systems. Expects candidates to sketch circuits on request.
Simon
Machinery & 2-Stroke
Main engine timing, VIT, scavenging, turbocharging, fuel system fault-finding. Links machinery to GHG/CII context. Systematic step-by-step approach expected.
Senthil
Safety / LSA / MOB
MOB recovery, enclosed space rescue, LSA functional tests and PMS records. Tests whether candidates know the paperwork behind the procedure, not just the procedure.
