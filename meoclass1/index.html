What's on this page & how to use the Cheat Sheets

Structure: Questions are organised into 9 topic clusters (QB1–QB9), matching the Kochi MMD oral syllabus. As new WhatsApp oral reports come in, they're added as companion files under the same QB number — e.g. QB1_C, QB1_D, QB1_E are all Stability/Construction/Survey questions, just added in later batches. The letter does not mean difficulty or priority — treat every letter under a QB number as equally important.

Cheat Sheets: A Cheat Sheet is a print-ready, 15-second-answer version of a QB file — same questions, condensed to key numbers and regulation hooks for rapid recall before your oral. Where a QB file has 6 or fewer questions, it does not get a separate Cheat Sheet file — the compressed answers are built directly into that QB file itself, so open the QB file and you already have both.

Search: Typing in the search box above checks both QB titles and the actual question text inside every QB file. If your search matches a specific question, you'll see which QB file it's in — the answer itself stays behind the subscriber gate.
