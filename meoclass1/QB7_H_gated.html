Q1
What safety measures are required for using low flash point fuel onboard?
ghgmachineryExaminer: Simon
15-Second AnswerShips using gas or other low-flashpoint fuels (flashpoint below 60°C — LNG, methanol) must comply with the IGF Code, mandatory under SOLAS Ch II-1 Part G for ships built on/after 1 January 2017. It sets goal-based requirements for fuel containment, double-barrier piping, gas-safe machinery spaces, ventilation, gas/fire detection, and ESD systems.
60-Second AnswerShips using gases or other low-flashpoint fuels (flashpoint below 60°C, e.g. LNG and methanol) must comply with the IGF Code, made mandatory via SOLAS amendments — Part F provides an alternative-design methodology for machinery/electrical/fuel storage systems, and a new Part G requires ships built on/after 1 January 2017 to comply with the IGF Code, with related SOLAS II-2 fire-safety amendments. The Code aims to minimise risk to ship, crew and environment, taking a goal-based approach with functional requirements per section. Key measures: double-walled/double-barrier fuel piping in gas-safe machinery spaces; fuel tanks designed to contain a leak; mandatory gas and fire detection with automatic ESD on fuel supply; specified ventilation rates for machinery spaces and tank connection spaces; hazardous area zoning; bunkering safety procedures with PPE for cryogenic fuels; and STCW training requirements for gas-fuelled ship crew.

Governing Instrument

IGF Code, mandatory under SOLAS Ch II-1 Part G (new ships from 1 Jan 2017) and related SOLAS Ch II-2 fire safety amendments; the IGC Code applies instead for gas carriers using cargo as fuel.

Design Safety Measures

  • Fuel containment: tanks designed for the specific fuel (cryogenic for LNG), protected against collision damage per IGF Code location requirements
  • Double-barrier / gas-safe machinery spaces: double-wall pipe-in-pipe or equivalent through machinery spaces, annular space leak-monitored; machinery space designed "gas-safe"
  • Ventilation: mandatory rates for fuel preparation rooms, tank connection spaces, any leak-accumulation space
  • Hazardous area classification: zoning around fuel tanks, bunker stations, vent outlets

Detection & Emergency Systems

  • Fixed gas detection in machinery/tank connection spaces and ducts, wired to alarm and automatic ESD
  • Fire detection integrated with SOLAS II-2, adapted for the fuel's fire characteristics (e.g. cryogenic pool fire for LNG)
  • Emergency Shutdown (ESD) system — automatic isolation on gas leak/fire/loss of ventilation, plus manual pushbuttons

Operational & Training Measures

  • Bunkering safety checklist (ship-shore/ship-ship), compatibility check, static electricity precautions
  • PPE for cryogenic fuel handling
  • STCW training/certification for gas-fuelled ship officers and crew

Key Numbers / Regs

  • Flashpoint threshold defining 'low-flashpoint fuel': below 60°C
  • IGF Code mandatory: ships built on/after 1 January 2017
  • SOLAS II-1 Part F (methodology) / Part G (mandatory application)

Deep-Dive — CE Relevance, Traps, Failures, Numbers, Casualty, Examiner Chain, On My Vessel
Chief Engineer Relevance

As CE on a gas/methanol dual-fuel vessel you're responsible for ESD system testing per PMS, gas detection calibration, ventilation fan checks in fuel preparation rooms, and bunkering safety procedure execution — core CE competency on newer tonnage.

Trap Questions

Q: 'Does the IGF Code apply to gas carriers using cargo as fuel?' A: No — covered instead by IGC Code Ch 16, though the safety philosophy is similar.
Q: 'What flashpoint threshold defines a low-flashpoint fuel?' A: Below 60°C.

Common CE Failures

Confusing IGF Code (general low-flashpoint fuel ships) with IGC Code (gas carriers using cargo as fuel); not knowing the 1 Jan 2017 mandatory date; describing safety features vaguely without naming double-barrier/gas-safe machinery space concepts.

Numbers to Memorise

<60°C = low-flashpoint fuel threshold; 1 Jan 2017 = IGF Code mandatory date; SOLAS II-1 Part F/Part G.

Casualty Link

As of mid-2026 there is no widely reported major safety casualty specific to modern methanol dual-fuel container propulsion systems — the technology is early-stage (pioneered by Laura Mærsk, in service since late 2023). Operational discussion so far centres on bunkering standardisation and fuel sourcing rather than safety incidents. The more defensible oral answer is to state plainly that the safety case for these vessels currently rests on strict IGF Code compliance, classification society Alternative Design approval, and proactive near-miss reporting rather than a mature body of casualty case law — an honest, current answer that shows awareness of where the framework is still maturing.

Below 60°C — methanol (~11-12°C) and LNG both qualify.
▸ Which SOLAS chapter/part makes the IGF Code mandatory, and from when?
SOLAS Ch II-1 Part G — mandatory for ships built on/after 1 January 2017. Part F provides the alternative-design methodology.
▸ Does the IGF Code apply to gas carriers using cargo as fuel?
No — that's covered by IGC Code Chapter 16 instead, though the safety philosophy is similar.
▸ Key IGF Code safety layers (name at least 4)?
Double-barrier/gas-safe machinery spaces, fixed gas + fire detection, automatic ESD, ventilation of fuel prep/tank connection spaces, hazardous area zoning, bunkering safety procedures.
