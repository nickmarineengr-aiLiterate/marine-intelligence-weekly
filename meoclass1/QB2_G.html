Can you carry timber/log deck cargo on your vessel, and what precautions are taken?
FIRELSA

15-Second Answer

A container ship can't carry timber deck cargo unless it applies TDC Code stability guidance and holds a Timber Load Line — otherwise it's not permitted, regardless of deck space.

60-Second Answer

Timber/log deck cargo carriage is governed by the 2011 TDC Code, non-mandatory IMO guidance under Resolution A.1048(27), applicable to ships ≥24m. It requires enhanced intact stability per the 2008 IS Code — hard criteria of GM0≥0.10m, GZ area≥0.08 m·rad to 40°, max GZ≥0.25m at ≥25°, plus a water-absorption weight allowance (roughly 10–20% across sources, timber-type/vessel-size dependent, not one fixed figure) — a Timber Load Line assignment, proper athwartship/longitudinal lashing using wire ropes, chains and turnbuckles/tensioners per the Cargo Securing Manual, clear freeing ports and safe walkways over the stow, and fire precautions since timber and dunnage are combustible. A standard container vessel without TDC Code compliance and Timber Load Line assignment would not carry timber deck cargo as routine trade — I'd state that limitation plainly rather than describe carriage I'm not certified for.

Applicability

The 2011 TDC Code (Code of Safe Practice for Ships Carrying Timber Deck Cargoes), adopted by IMO Resolution A.1048(27) in 2011 (superseding the 1991 edition under A.715(17)), applies to all ships 24m or more in length carrying timber deck cargo. It is non-mandatory guidance — Administrations recommend it as the basis for national safety standards — but compliance with its stability and securing criteria is what allows a ship to be assigned and use Timber Load Line marks under the applicable Load Line Convention rules. A standard container vessel is not normally equipped or certified for this trade; carriage would require prior class/flag review of the specific stability case before loading.

Precautions when carried

  • Stability: the pre-loading plan must be checked against the ship's Trim and Stability Book, accounting for weight gain in the timber from water absorption over the voyage — a percentage allowance calculated per 2008 IS Code guidance (secondary sources cite figures roughly in the 10–20% range depending on timber type and vessel size; there is no single fixed percentage in the Code text itself, so verify the applicable figure against the vessel's own stability booklet), maintaining stability within required limits for the entire voyage. The 2008 IS Code also fixes hard numeric criteria for timber deck cargo ships: GM0 not less than 0.10m after corrections, GZ curve area ≥0.08 m·rad up to 40° heel (or angle of flooding if less), and maximum GZ ≥0.25m occurring at ≥25° heel.
  • Stowage & securing: lashed athwartships and secured longitudinally using certified wire ropes, chains, and turnbuckles/tensioners; uprights fitted for loose sawn wood; stoppers/low uprights to prevent sliding off hatch covers, per the Cargo Securing Manual and TDC Code Chapters 2-4 design principles.
  • Freeing ports & lifelines: kept clear/functional; safe walkways rigged over the deck cargo for crew access.
  • Fire risk: timber/dunnage is combustible — fire wires, hose reach, and detection near the stow checked; smoking restrictions enforced.
  • Freeboard: Timber Load Line marks used only when TDC Code stability/freeboard conditions are fully satisfied and the load line has been properly assigned.

CE Oral Tip: If your vessel has no Timber Load Line, say so plainly — examiners probe whether candidates falsely claim capability. State the TDC Code's non-mandatory-but-widely-applied status correctly (guidance the Administration adopts as its standard, not itself a Convention), name Resolution A.1048(27) if pressed, and quote the hard 2008 IS Code numbers (GM0 0.10m, GZ area 0.08 m·rad, max GZ 0.25m) rather than a specific water-absorption percentage, since that figure varies by source.
Deep-Dive: CE Relevance

CE ensures Cargo Securing Manual lashing gear (wire ropes, chains, turnbuckles) is certified and freeing ports are not obstructed by the stow — both are PSC/Class survey items if the ship ever does load timber deck cargo.

Deep-Dive: Trap Questions

“Is a container ship allowed to carry timber on deck?” — Only if TDC Code-compliant stability/freeboard is applied and a Timber Load Line assigned; otherwise no, regardless of physical deck space.

Deep-Dive: Common CE Failures

Confusing “general deck cargo lashing” rules with the specific TDC Code stability/water-absorption/freeboard regime — these are not interchangeable. Also: calling the TDC Code “mandatory” when it is formally non-mandatory guidance (though widely adopted), and stating a single universal water-absorption percentage as fixed fact when it varies by source/timber type.

Deep-Dive: Numbers to Memorise

TDC Code applies to ships ≥24m length; adopted by IMO Resolution A.1048(27) in 2011, superseding the 1991 edition (A.715(17)). Hard 2008 IS Code criteria: GM0 ≥0.10m after corrections, GZ curve area ≥0.08 m·rad to 40° heel, max GZ ≥0.25m at ≥25°. Water-absorption allowance: calculated per 2008 IS Code guidance, cited roughly 10–20% across sources — vessel/timber-type dependent, not one fixed number.

Deep-Dive: Casualty Link

Historical timber cargo shifting incidents drove the Code's revisions in 1978, 1991, and 2011, progressively tightening lashing and stability guidance.

Deep-Dive: On My Vessel (Maersk A/S, container)

On a Maersk container ship, timber deck cargo is not normal trade; any ad-hoc project cargo request would need charterer/class approval and a dedicated stability check against the TDC Code before loading.

Why is the rescue boat typically located on the starboard side of the vessel?
LSA

15-Second Answer

There's no SOLAS rule fixing the rescue boat's side — starboard is convention, historically rooted in right-handed propeller walk favouring port-side-to berthing, and reinforced by port-side terminal infrastructure (quay, STS cranes, gangway) obstructing that side once alongside.

60-Second Answer

SOLAS III/31 and the LSA Code don't specify which side the rescue boat must be on — this is a design and operational convention, not a regulatory requirement. Starboard is common for two compounding reasons. First, historically: single-screw ships with a right-handed propeller experience propeller walk that kicks the stern to port when going astern, making port-side-to berthing the natural, self-correcting approach long before container terminals existed — though modern large container vessels rely more on tugs and thrusters than this effect in practice. Second, operationally: because ships berth port-side-to by convention, the port side becomes obstructed by the quay, fenders, mooring lines, an actively working ship-to-shore gantry crane, and the rigged accommodation ladder, leaving starboard as the clear, seaward side for immediate rescue boat deployment. Ship-specific arrangement plans can vary, so I'd always confirm which side my own vessel uses and why during handover.

Regulatory basis — no side is mandated

SOLAS Chapter III, Regulation 31 (cargo ships) requires at least one rescue boat complying with LSA Code section 5.1. The regulation governs number, readiness (launch within 5 minutes, under trim up to 10° and list up to 20° either way per LSA Code 5.1.1.1 / SOLAS III/13), suitable position, and non-interference with other survival craft — it says nothing about port or starboard.

Why starboard became the convention

  • Port-side-to berthing: most ports and terminals are arranged for standard port-side alongside berthing. Note: modern vessels, including Maersk container ships, carry fully replicated cargo/bunker manifolds on both sides for berthing flexibility — the manifold itself is not the driver of this convention.
  • Hydrodynamic root cause: this convention traces back to single-screw ships with a right-handed (clockwise, viewed from astern) propeller. When astern propulsion is applied to kill headway on final approach, propeller walk — driven by the denser, less turbulent water acting on the lower blades versus the wake-affected upper blades — kicks the stern to port. That makes a port-side approach self-correcting: the same astern burst that stops the ship also tucks the stern in against the quay, whereas a starboard-side approach fights this effect. This is why port-side-to became the default long before container terminals existed.
  • Modern caveat: large container vessels like a Maersk boxship typically berth with tug assistance and bow/stern thrusters, so this hydrodynamic tendency is no longer the deciding factor in practice — but it remains the historical/doctrinal basis for the convention, and is worth citing as the "why" behind the "why" if the examiner probes further.
  • Real hazard once alongside: when berthed port-side-to, the port side becomes a working industrial zone — quay wall, fenders, mooring lines, an actively slewing ship-to-shore (STS) gantry crane overhead, and the accommodation ladder/gangway rigged out. Launching a rescue boat into the narrow gap between hull and jetty, under a moving crane, is physically hazardous or impossible.
  • Starboard stays clear: starboard remains the open, seaward side — a clean drop into open water, and no conflict with the gangway's swing-out radius or falls.
  • Design consequence: this is also why most GA drawings conventionally show the starboard profile — it's the side carrying the rescue boat and davit-launched liferaft.

Exception — passenger ships

SOLAS III/21 requires passenger ships of 500 GT and over to carry one rescue boat on each side, reflecting their larger scale and the need for redundancy regardless of which side is alongside.

CE Oral Tip: State plainly: 'the regulation does not mandate a side — this is operational convention.' If pressed on WHY port-side-to berthing became standard, cite propeller walk (right-handed prop, astern kicks stern to port) as the historical/doctrinal root — verified textbook shiphandling knowledge, safe to state confidently. If pressed on the manifold angle, acknowledge modern ships have manifolds on both sides — the immediate driver of the convention is terminal infrastructure (STS crane, gangway, quay), not the manifold. Never invent a pilot-ladder or COLREGs Rule 9 link — those don't hold up under questioning.
Deep-Dive: CE Relevance

CE verifies during monthly drills (SOLAS III/19.3.3.6) that the rescue boat, wherever fitted, launches within 5 minutes and that davit/fall gear is unobstructed by cargo lashings, gangway rigging, or STS crane reach on that side.

Deep-Dive: Trap Questions

“Which SOLAS regulation fixes the side?” — none does; naming a specific side-mandating regulation is the trap answer. Cite III/31 (cargo ships, no side) correctly and explain the berthing-convention logic separately. Also: if pressed on manifolds, don't claim they're port-side only — modern ships replicate both sides.

Deep-Dive: Common CE Failures

Confusing SOLAS III/21 (passenger ship each-side requirement) with the general cargo ship rule in III/31 — these apply to different ship types.

Deep-Dive: Numbers to Memorise

Rescue boat must be capable of being launched and waterborne within 5 minutes, under trim up to 10° and list up to 20° either way (LSA Code 5.1.1.1, SOLAS III/13). Monthly launch with assigned crew, minimum once every 3 months (SOLAS III/19.3.3.6). Right-handed propeller = clockwise viewed from astern in ahead gear; astern kicks the stern to port (propeller walk) — verified standard shiphandling doctrine.

Deep-Dive: Casualty Link

Not casualty-driven — this is a design/operational convention answer rather than a lessons-learned regulation.

Deep-Dive: On My Vessel (Maersk A/S, container)

On the Maersk container ship, confirm during handover which side the vessel typically berths, and note that the accommodation ladder and active STS crane reach are on that side while alongside — the practical reason the rescue boat sits on the opposite (usually starboard) side.
