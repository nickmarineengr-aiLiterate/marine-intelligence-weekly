1
Can an LNG carrier carry chemical cargo?
GAS CARRIER

15-Second Answer

Not under a standard IGC-only certificate. A vessel must be dual-certified — holding both an IGC Code Certificate of Fitness and an IBC Code Certificate of Fitness — and the specific chemical must be listed on the IBC certificate, with containment materials/structure compatible with its density and chemistry.

60-Second Answer

Under SOLAS Chapter VII, a standard LNG carrier is designed and certified only under the IGC Code (Res. MSC.370(93)), for cryogenic, low-specific-gravity cargo (~0.45–0.5). Bulk liquid chemicals fall under the IBC Code (Res. MSC.369(93)), which is a separate certification regime. Some substances — ammonia, ethylene oxide/propylene oxide — are liquefied gases that are also chemically hazardous and are explicitly cross-referenced in IGC Chapter 19, so they can be carried on a gas carrier if listed on its Certificate of Fitness. But true bulk chemicals (methanol, toluene, acids) cannot be loaded into a standard LNG carrier: the tanks, pumps and materials are not approved or documented for that duty, and membrane tanks are structurally rated for low density, not S.G. 0.8–1.5 chemical cargoes.

Governing instruments

SOLAS Chapter VII; IGC Code (Res. MSC.370(93)) for liquefied gases; IBC Code (Res. MSC.369(93)) for bulk chemicals.

Core concept

IGC and IBC are separate certification regimes. A gas carrier only becomes chemical-capable if it is a "dual-code" ship holding both a valid IGC and a valid IBC Certificate of Fitness for the specific cargo.

Key limiting factors

  • Specific gravity: LNG containment (esp. membrane) is structurally rated for S.G. ~0.5; chemical cargoes typically run S.G. 0.8–1.5+, requiring separate structural approval.
  • Material compatibility: pumps, piping, valves and gaskets must resist the specific chemical; LNG submerged pumps may lack torque/clearance for denser or viscous products.
  • Vapour/BOG handling: chemicals don't boil off like LNG but may need vapour-return lines or inert-gas blanketing per IBC Code Ch.9.
  • Tank changeover: switching cargo types needs full warm-up, gas-freeing, inspection and possibly chemical washing — standard LNG carriers lack slop tanks/wash-water processing.

Trap point

Ammonia, VCM, ethylene/propylene oxide are chemically hazardous liquefied gases cross-listed in IGC Chapter 19 — they CAN go on a gas carrier if the Certificate of Fitness lists them. That is different from true bulk chemicals, which cannot.

CE Oral Tip: Senthil tests structural realism, not theory. Open with: "No, sir, not under a standard IGC-only vessel — it requires dual IGC and IBC Certificates of Fitness for that exact cargo." Then name the specific-gravity structural limit before anything else.
Deep-Dive: CE Relevance

On a dual-certified vessel the CE verifies containment materials against manufacturer resistance lists and manages distinct vapour/venting arrangements for toxic chemical vapours versus the FGSS used for LNG fuel.

Deep-Dive: Trap Questions

Q: "Aren't ammonia/VCM chemicals?" A: Yes, but as liquefied gases they're explicitly listed in IGC Ch.19 and can be carried on a gas carrier if the Certificate of Fitness names them and materials are compatible (e.g. avoiding copper–ammonia reaction). Q: "If S.G. matches at 0.5, can I load a chemical into an LNG membrane tank?" A: No — secondary-barrier chemical compatibility, pump lubrication (submerged pumps use cargo for cooling) and MARPOL Annex II slop-discharge rules still require a valid IBC Certificate of Fitness for that chemical.

Deep-Dive: Common CE Failures

Treating IGC and IBC as one regime instead of two distinct SOLAS Ch.VII certification tracks; forgetting the density/structural-rating argument; not distinguishing true bulk chemicals from IGC-listed liquefied-gas chemicals.

Deep-Dive: Numbers to Memorise

LNG boiling point −163°C; typical LNG S.G. ≈0.45–0.5 vs chemical S.G. 0.8–1.5+; IGC Code Chapter 19 = summary of minimum requirements listing cross-referenced cargoes.

Deep-Dive: On My Vessel

Not directly applicable — a container vessel carries dangerous goods only in packaged form under SOLAS VII Reg.4 and the IMDG Code, not in bulk liquid/liquefied-gas form under IGC/IBC.

2
IGC Code — explain Type A, B, and C tanks and membrane tanks: shape of each, which is self-supporting, and which has more carrying capacity.
GAS CARRIER

15-Second Answer

Types A, B and C are independent/self-supporting tanks; Membrane is not (it relies on the ship's hull via insulation). Type A is prismatic with a full secondary barrier; Type B (spherical/Moss or prismatic/SPB) uses a partial barrier on a leak-before-failure basis; Type C is a cylindrical pressure vessel needing no secondary barrier. Membrane gives the highest carrying capacity by eliminating dead space; Type C gives the lowest.

60-Second Answer

IGC Code Chapter 4 defines four containment types. Type A tanks are prismatic, self-supporting, low-pressure (<0.7 bar) and require a full secondary barrier able to hold the total cargo volume if the primary fails. Type B tanks are also self-supporting — spherical (Moss design) or prismatic (SPB) — but need only a partial secondary barrier (drip tray/splash shield) because of rigorous 'leak-before-failure' stress analysis; spheres cost carrying capacity through hull dead space. Type C tanks are self-supporting cylindrical, bilobe or spherical pressure vessels (>2 bar, up to ~18 bar) needing no secondary barrier at all, since catastrophic failure is engineered out — but they are heavy and have the lowest volumetric efficiency. Membrane tanks are the odd one out: not self-supporting, following the inner hull's orthogonal geometry with the secondary barrier built into the insulation layers, giving the highest cargo volume per hull dimension by eliminating dead space.

Governing instrument

IGC Code Chapter 4 (Res. MSC.370(93)), Sections 4.21–4.25 define the four containment systems.

Trap point

Type B is not always spherical — the SPB (prismatic) design is still Type B; the defining feature is the leak-before-failure stress analysis that permits a reduced (partial) secondary barrier, not the shape.

Deep-Dive: CE Relevance

On LNG-fuelled container vessels the CE manages Type C or Membrane fuel tanks for the FGSS — boil-off gas, inter-barrier gas detection, insulation vacuum spaces, and controlled cool-down rates during bunkering to limit thermal stress on the inner hull.

Deep-Dive: Trap Questions

Q: "Why doesn't Type C need a secondary barrier?" A: It's engineered as a true pressure vessel where crack propagation is negligibly slow and detectable long before failure, so the statutory secondary-barrier requirement is waived. Q: "Which tank type is most sloshing-sensitive?" A: Membrane — thin primary barrier and insulation are vulnerable to partial-fill liquid impact, so membrane tanks operate under strict filling-limit restrictions (typically below ~10% or above ~70–80% volume).

Deep-Dive: Common CE Failures

Stating membrane tanks are self-supporting (they are not); forgetting Type B needs a partial, not zero, secondary barrier; failing to state clearly that Membrane has the highest carrying capacity.

Deep-Dive: Numbers to Memorise

Type A/Membrane design pressure <0.7 bar; Type C design pressure >2 bar (up to ~18 bar); LNG cryogenic temperature −163°C; membrane sloshing-restricted fill zone roughly 10%–80%.

Deep-Dive: Casualty Link

Mid-2000s Mark III membrane sloshing incidents — several LNG carriers suffered primary-barrier deformation and insulation crushing from resonant sloshing at partial fill levels in heavy weather, reinforcing the strict filling-limit rules for membrane systems.

Deep-Dive: On My Vessel

If our vessel were LNG dual-fuelled, fuel would be held in a Type C or Membrane tank feeding the main engine via a high-pressure FGSS at roughly 300 bar; as a conventional container ship this is only indirectly relevant unless retrofitted with LNG fuel gas storage.
