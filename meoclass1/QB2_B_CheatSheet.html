⚠ IMDG & Dangerous Goods — Q1, Q3, Q4, Q5, Q8
Q1
IMDG Docs, Certificates & EmS/MFAG Location

Key Documents: Document of Compliance (DoC) — SOLAS II-2/19; Dangerous Goods Manifest (Stowage Plan); Dangerous Goods Declaration (DGD) — from shipper; MSDS — SOLAS VI/5-1.

Firefighting info location: IMDG Code Supplement → EmS Guide

EmS Guide Structure
  • Fire Schedules F-A to F-J — medium, strategy, approach
  • Spill Schedules S-A to S-Z — containment & toxic hazard mgmt
  • MFAG — Medical First Aid Guide (same supplement)
  • DoC specifies which holds can carry which classes
⚠ Trap"What book do you open first if Hold 3 emits yellow smoke?" → Get UN Number from DG Manifest, then open EmS Guide in IMDG Supplement for the specific Fire Schedule. Not "the IMDG Code" generically.
Regulatory
SOLAS II-2/Reg.19Structural certification to carry specific DG classes
SOLAS VII/Reg.4Mandatory Dangerous Goods Manifest
CE Oral TipNumbers: SOLAS II-2 Reg 19; 9 IMDG hazard classes; IMDG mandatory since 1 Jan 2004.
Q3
IMDG Marking, Labelling & Numbering

Marks on package: Proper Shipping Name (PSN) — no trade names; UN Number with "UN" prefix (e.g. UN 1993); 3-month sea immersion durability.

Labels on package: Diamond-shaped hazard placard — min 100 mm × 100 mm; colour-coded by class; secondary risk labels if dual hazard (no class number on secondary).

Labels on container exterior: Minimum 250 mm × 250 mm placards on all 4 sides.

Special marks: Marine Pollutant (fish+tree); Elevated Temperature (thermometer in triangle).

Key Numbers
  • 100 mm × 100 mm — label on individual package
  • 250 mm × 250 mm — placard on container exterior
  • 3 months — sea immersion durability threshold
  • Chapter 5.2 = package marking/labelling; Chapter 5.3 = container placarding
⚠ Trap"Difference between a label and a placard?" → Labels (100 mm) on individual packages; Placards (250 mm) on the outside of the shipping container. Different sizes, different purposes.
Q4
IMDG Code — Structure, Classes & Key Features

Structure: 2 Volumes + Supplement. Vol.1 = General provisions, classification, packing; Vol.2 = Dangerous Goods List (DGL); Supplement = EmS, MFAG, reporting forms.

9 Hazard Classes:

Classes
  • Class 1 Explosives · Class 2 Gases · Class 3 Flammable Liquids
  • Class 4 Flammable Solids/Reactive · Class 5 Oxidizers/Peroxides
  • Class 6 Toxic/Infectious · Class 7 Radioactive · Class 8 Corrosives
  • Class 9 Miscellaneous (inc. Li-Ion batteries, dry ice)
Key Features
  • Mandatory under SOLAS since 1 January 2004
  • UN Number = 4-digit hazard identifier (look up in DGL)
  • Packing Groups I/II/III = degree of danger
  • Segregation codes = minimum separation between incompatibles
  • Amended every 2 years (biennial cycle)
CE Oral TipNair will ask class numbers. Know: 4.3 = water-reactive (no water!); 5.1 = oxidizers (accelerates fire); 6.1 = toxic. These three drive firefighting approach changes.
Q5
Mislabelled Container Fire — Wan Hai 266

Wan Hai 266 (June 2021, off Cochin): Container fire involving undeclared/mislabelled chemical cargo — forced Cochin coast guard intervention, vessel drifted close to Indian coast.

Action if mislabelled container fires:

CE Response Sequence
  • Cannot trust declared UN number → treat as unknown/most hazardous
  • Heavy boundary cooling to adjacent containers — prevent thermal spread
  • Deploy portable water monitors from upwind/safe distance
  • Isolate reefer power rails in affected bay
  • Notify Master + company DPA immediately; prepare GMDSS distress readiness
  • Do NOT open container doors — risk of backdraft or toxic release
⚠ TrapNever attack mislabelled chemical fire at close range with water — Class 4.3 water-reactive materials create explosive hydrogen gas. Boundary cool only until UN number confirmed via manifest.

EmS F-G / F-HFire schedules for toxic/reactive cargo
Q8
UN Number → Hazard Identification & Firefighting

Process: Get UN Number from DG Manifest → look up in IMDG DGL (Vol.2) → column 15 gives EmS Fire Schedule code → open IMDG Supplement EmS Guide.

EmS Code → Firefighting Medium
  • F-A: Water (general use) — most cargo fires
  • F-G: Water spray OK but avoid direct jet
  • F-H / 4.3 hazard: NO WATER → DCP, CO₂, inert gas, isolation only
  • F-J (Class 1 Explosives): Do not fight — abandon and evacuate
Key Numbers
  • 4-digit UN number (e.g. UN 1993 = flammable liquid NOS)
  • Column 15 in DGL = EmS code assignment
  • Class 4.3 (water-reactive) → hydrogen explosion risk from water
CE Oral TipExaminer will name a class and ask firefighting medium. Key rule: Class 4.3 = never water. Class 3 flammable liquid = foam/CO₂ for pool fires, water for boundary cooling. Always EmS first, then decide.
🛢 Cargo Types & Codes — Q6, Q14, Q15
Q6
LPG Hazards on a Gas Carrier (MAIB Learnings)

LPG key properties: Heavier than air (pools in low spaces); flammable range 1.8–9.5% (propane); liquefied under pressure at ambient temp or refrigerated at atmospheric pressure; boil-off gas (BOG) constant management required.

Critical hazards:

Hazard Categories
  • Fire/Explosion: Vapour cloud ignition; BLEVE (Boiling Liquid Expanding Vapour Explosion) if pressurised tank exposed to fire
  • Cryogenic burns: Liquid LPG contact causes severe frostbite
  • Asphyxiation: BOG displaces oxygen in enclosed spaces
  • Expansion ratio: 1 litre liquid → approx. 250 litres vapour
Key Numbers
  • Expansion factor: 1:250
  • Governing Code: IGC Code Chapter 13 and 18
  • Moritz Schulte (MAIB 2021): secondary fuel system oversight → catastrophic fire risk
CE Oral TipMAIB investigation citation shows examiner you read beyond the code. Key lesson: gas carrier incidents often involve auxiliary/secondary machinery interfaces, not primary cargo systems.
Q14
IBC Code vs BCH Code — Chemical Tankers

The split: Determined purely by construction date — 1 July 1986.

BCH Code (pre-1986)
  • Older chemical tankers built before 1 July 1986
  • Pragmatic safety rules — cannot be retrofitted to modern IBC standards
  • Phasing out as fleet ages
IBC Code (post-1986)
  • Chemical tankers built on or after 1 July 1986
  • Three ship types: Type 1 (highest hazard, max survivability), Type 2, Type 3 (least hazardous)
  • Advanced damage stability, double-hull spacing, tank location rules
  • Mandatory under SOLAS VII/Reg.8 + MARPOL Annex II/Reg.11
⚠ Trap"Ship laid down 1980, major conversion 1990 — which code?" → Major structural modification can cause class to classify it as a new build → full IBC Code applies. Cannot load substances beyond Certificate of Fitness regardless of code.
Q15
IMSBC Code — Groups A, B, C

Mandatory under SOLAS VII/Reg.7-1. Covers solid bulk cargoes only (excludes grain — SOLAS VI).

Three Groups
  • Group A — Liquefiable: Fine particles + moisture → liquefaction under vibration → sudden stability loss. Examples: iron ore fines, nickel ore, bauxite.
  • Group B — Chemical Hazards (MHB): Self-heating, oxygen depletion, toxic gas emission, corrosive. Examples: coal, direct reduced iron (DRI), fertilisers.
  • Group C — Neither: Physically stable but may shift if untrimmed or overload tank top. Examples: clinker, salt, stone aggregates.
Key Numbers
  • TML = 90% of FMP (Flow Moisture Point)
  • Angle of repose marker: 35°
  • Cannot load Group A if moisture content > TML
  • Casualties: MV Bulk Jupiter (2015) — bauxite liquefaction sinking
⚠ Trap"Chief Officer wants to wash down Group A hold during rainy port stay." → Refuse. Introducing water can push moisture above TML → liquefaction risk at sea → capsizing.
📦 Container Ship Operations — Q2, Q7, Q9, Q11, Q12, Q16
Q2
Cargo Securing Arrangement & Safe Access Plan (CSAP)

CSM (Cargo Securing Manual): Mandatory, vessel-specific, approved by Class/Administration under SOLAS VI/Reg.5. Dictates stack weights, lashing patterns, gear certification.

CSAP (Cargo Safe Access Plan): Embedded within CSM. Provides dedicated walkways, permanent lashing platforms with handrails, safe ladder transitions, designated fall-protection anchor points.

Key Numbers
  • SOLAS VI/Reg.5.6 — cargo stowed per CSM
  • CSS Code Annex 14 — safe working conditions for lashing
  • 5-year proof-load testing for loose lashing gear
  • Lashing Gear Register = official inventory of loose gear certificates
⚠ CE Trap"Sheared lashing bridge gusset — C/O wants to load now." → Refuse. Structural damage invalidates CSM for that bay. Class-approved repair required before loading. Lashing is deck dept but structural repairs = CE jurisdiction under PMS.
Q7
Liner / VOCC / NVOCC Model

Liner Service: Fixed published schedule, designated port rotations, standardised tariffs. The backbone of container shipping.

VOCC vs NVOCC
  • VOCC (e.g. Maersk): Owns/operates vessels. Issues Master Bill of Lading. Direct control over speed, routing, terminal priority. High capital cost.
  • NVOCC: Buys space in bulk from VOCC. Issues House Bill of Lading to shippers. Asset-light, flexible, can consolidate LCL. Zero control over vessel delays.
CE Commercial Link
  • Liner schedules are calculated to the hour — ER delays cost berth slots
  • Missed terminal window → demurrage + NVOCC contract penalties on VOCC
  • MBL (Master B/L) = VOCC liability; HBL (House B/L) = NVOCC liability
CE Oral TipNair: "Why maintain precise charter speed even when fuel is expensive?" → Schedule integrity is paramount — missed port windows break NVOCC commitments and generate claims.
Q9
CSC — PES & ACEP Inspection Schemes

CSC (International Convention for Safe Containers): Every container must carry a durable CSC Safety Approval Plate on left door. Two inspection pathways:

PES — Periodic Examination Scheme
  • First inspection: within 5 years of manufacture date
  • Subsequent: every 30 months
  • NED (Next Examination Date) stamped on CSC plate
ACEP — Approved Continuous Examination Programme
  • Inspection integrated into depot maintenance cycles — no fixed calendar
  • ACEP registration stamp on plate instead of NED (e.g. ACEP-DK-2026)
  • More frequent than 30-month PES in practice
  • Preferred by large operators like Maersk
Tests Carried Out (ISO 1496)
  • Floor strength, corner casting integrity, stacking load, racking, door seals/latches
⚠ TrapACEP containers have no NED stamped on the plate. If a PSCO sees no NED, they check for ACEP stamp — not automatically deficient.
Q11
VGM — Verified Gross Mass

SOLAS VI/Reg.2 — mandatory from 1 July 2016. Shipper must declare total certified weight of packed container before loading.

VGM = Cargo mass + Dunnage/securing gear + Container tare weight
Two Approved Methods
  • Method 1: Weigh fully loaded sealed container on certified weighbridge
  • Method 2: Weigh each item + dunnage individually, add printed tare weight. Requires certified equipment + authority audit.
Key Rules
  • No VGM = no load — strict prohibition under SOLAS
  • Signed declaration submitted to Master + terminal before loading
  • Calibration of weighbridge = shipper's/state authority responsibility — NOT crew
  • MSC Napoli (2007) — misdeclared weights drove VGM mandate
⚠ Trap"Crane operator says container feels heavier than VGM." → Master has overriding authority to refuse loading until weight is re-verified.
Q12
GT & NT — ITC 1969 Formulas

ITC 1969 (International Convention on Tonnage Measurement of Ships). GT and NT are dimensionless numbers — no units.

GT = K₁ × V where K₁ = 0.2 + 0.02 log₁₀(V) NT = K₂ × Vc × (4d/3D)² + K₃(N₁ + N₂/10) where K₂ = 0.2 + 0.02 log₁₀(Vc)
Variables
  • V = total enclosed moulded volume (m³); Vc = cargo space volume
  • d = moulded draught at summer load line; D = moulded depth
  • N₁ = berths for passengers in cabins ≤8; N₂ = other passengers
  • Constraint: (4d/3D)² ≤ 1; NT ≥ 0.30 × GT
Key Facts
  • Deck containers (above hatch covers) excluded from standard Vc — but Suez (SCNT) and Panama (PC/UMS) use own tonnage systems including deck stacks
  • "d" in NT formula is the fixed summer load line draught — NT does NOT change with actual operating draught
⚠ Trap"Does NT change when ship is in light ballast?" → No. The "d" used is fixed at design summer load line draught stamped on the Tonnage Certificate.
Q16
Container Test Procedures (Full Range)

Governed by ISO 1496-1 and CSC Convention Annex I. Tests go well beyond the one-door-off test.

Structural Tests
  • Stacking test: 8 × tare + maximum payload on corner posts
  • Floor strength test: Forklift with 5,460 kg on each fork
  • Roof strength test: 300 kg evenly distributed
  • Racking test: 15,240 kg lateral shear force applied diagonally
  • End-wall strength: 0.4 × rated payload; side-wall = 0.6 × rated payload
Leakage / Weathertight Tests
  • Hose test: Water at 100 kPa (1 bar) nozzle pressure; 100 mm nozzle; ≤3 m distance — checks door seals & gaskets
  • Ultrasonic test (non-destructive): For clean cargo — transmitter inside, receiver scans exterior seams
  • Chalk test: Rubber compression check only — NOT a structural test
Key Numbers
  • Floor test: 5,460 kg; ISO standard: ISO 1496-1
🔥 Fire Safety & FSA — Q10, Q17, Q18, Q19
Q10
Portable Water Monitors — SOLAS Amendment corrected

MSC.365(93) → SOLAS II-2/Reg.10.7.3 — applies to ships built on/after 1 Jan 2016.

The 5-Tier Rule
  • ≥ 5 tiers of containers on/above weather deck → minimum 2 portable water monitors mandatory
  • < 5 tiers → no portable monitors mandated (standard hydrants baseline)
Key Numbers — CORRECTED
  • Each monitor min flow: 400 L/min (24 m³/hr) — per SOLAS II-2/10.7.3.1
  • Two monitors combined: 800 L/min (48 m³/hr)
  • Well within single main fire pump capability (~80–120 m³/hr on large container ships)
  • Min hydrant pressure maintained: 0.27 N/mm²
  • Do NOT quote 80 m³/hr — that was an error in the draft source
⚠ Trap"Can the emergency fire pump run two portable monitors?" → Not recommended — emergency pump is sized for ~25–30 m³/hr (two standard jets). For monitors + safety lines, run main fire pumps.
CE Oral TipNair: "Does a portable monitor replace standard fire hoses?" → No. Monitors are in addition to standard hydrants/hoses. Supplementary high-volume tool for high stack access.
Q17
Fixed Deck DCP System

Governed by SOLAS II-2 + FSS Code Chapter 11. Protects weather deck zones where flammable gas (Class 2) or low-flashpoint liquid (Class 3) containers are carried under the DoC.

System Components
  • Pressurised dry chemical powder storage vessels
  • High-pressure nitrogen (N₂) propellant cylinder manifold
  • Pilot activation lines + distribution manifold
  • Fixed deck monitors + hand hose reels
Key Numbers
  • Minimum discharge duration: 45 seconds from all monitors simultaneously
  • Powder must remain free-flowing — monthly N₂ pressure checks; annual powder fluffing
  • Do NOT discharge powder during testing — use dry compressed air/N₂ blow-through of deck lines
⚠ Trap"How do you test DCP lines during monthly inspection?" → Do not discharge powder. Verify N₂ pressure gauges, check free-flow, blow deck piping with dry compressed air/nitrogen with drainage valves open.
Q18
Deck Firefighting Procedure — Confirmed/Suspected Fire

Container fire = long duration, heat-intensive, toxic smoke. Strategy = boundary cooling first, not direct attack.

CE Action Sequence
  • 1. Sound alarm + muster fire party; notify bridge/Master
  • 2. Start main fire pump(s) — do not rely on emergency pump alone
  • 3. Identify cargo — check DG Manifest for UN number, EmS code
  • 4. Deploy portable water monitors upwind for boundary cooling of adjacent containers
  • 5. Do NOT open container doors — backdraft/toxic release risk
  • 6. Isolate reefer power rails in affected bay
  • 7. Monitor stability — firefighting water accumulation affects GM
  • 8. Call MRCC/Coast Guard if beyond crew capability
Key Numbers
  • Maintain 0.27 N/mm² at all hydrants during firefighting
  • If Class 4.3 confirmed → no water on source; boundary cool steel only
  • Monitor GM continuously — water on deck = free surface effect
CasualtyHistorical CTL fires: delayed boundary cooling allowed thermal energy to spread — total loss from a small initial fire. Boundary cool early and aggressively.
Q19
FSA for Li-Ion Battery Cargo (MSC.1/Circ.1647)

FSA = Formal Safety Assessment. IMO-approved 5-step proactive risk methodology. Applied here per MSC.1/Circ.1647 interim guidelines for Li-ion containers.

5 Steps
  • Step 1 — HAZID: Identify thermal runaway triggers, HF gas off-gassing, explosive venting
  • Step 2 — Risk Assessment: Probability × consequence matrix; likelihood of cascading thermal runaway
  • Step 3 — Risk Control Options (RCOs): Segregation from heat boundaries; hold gas detection; portable water monitors; temperature monitoring
  • Step 4 — Cost-Benefit Analysis: Evaluate cost of RCOs vs. financial/environmental damage of an incident
  • Step 5 — Recommendations: Formal regulatory proposals to IMO/flag state for new amendments
Key Numbers
  • 5 FSA steps — must know in sequence
  • MSC.1/Circ.1647 — interim Li-ion cargo guidelines
  • HF (hydrogen fluoride) gas = primary toxic off-gas during thermal runaway
⚠ Trap"Why FSA instead of just using the IMDG Code?" → IMDG is reactive (updated after accidents); FSA is proactive — analyses risk and builds defences before an incident occurs.
🔧 Construction & Integrity — Q13
Q13
Watertight vs Weathertight — Definitions & Test Methods
Watertight
  • Resists continuous head of water pressure from both sides
  • Location: Below/at bulkhead deck — collision bulkheads, tank boundaries, WT doors, double bottom
  • SOLAS II-1 structural requirement
Weathertight
  • Prevents water penetration from transient external exposure only (spray, green water)
  • Cannot withstand sustained submersion
  • Location: Above waterline — hatch covers, superstructure doors, spurling pipes
  • ICLL Regulation 16
Test Methods
  • Watertight — Hydrostatic hose test: High-pressure jet; standard nozzle 12.5 mm; min 0.2 MPa (2 bar) nozzle pressure; ≤1.5 m from surface
  • Watertight — Ultrasonic (UTT): Non-destructive; transmitter inside, receiver outside seams; preferred for cargo spaces
  • Weathertight — Low-pressure hose test: Standard hose, 2 bar; checks gasket compression
  • Weathertight — Chalk test: Rubber gasket compression check only — NOT valid for structural WT survey
⚠ Class Survey Trap"Can a chalk test pass a Class survey for a repaired WT collision bulkhead?" → No. Chalk test = weathertight gasket check only. Repaired WT boundary needs high-pressure hydrostatic hose or Class-approved UTT.
